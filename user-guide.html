User Guide

  1. Sign Up / Log In: Use an email to create an account (prototype).
  2. Add Offers: Enter signup bonuses or pick from a catalog.
  3. Track Tasks: Check off DDs, card swipes, and balance rules.
  4. Enable Alerts: Note upcoming deadlines and milestones.
  5. Review Analytics: See earned totals and projected rewards.

Tip: Check your dashboard weekly to avoid last minute scrambles.
