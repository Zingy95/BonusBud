Application Prototype Information

BonusBud is a simple way to manage bank signup bonuses without linking bank accounts. Track tasks, due dates, and payouts in one place.

Tech stack

  • HTML and CSS for structure and design
  • Vanilla JavaScript for interactions
  • GitHub Pages for hosting

Core modules

  • Offer catalog and custom offers
  • Requirement checklist with due dates
  • Deadline alerts and reminders
  • Basic analytics overview

Data model

User to Offers to Tasks. Each task has a type, a due date, and a completion status.
