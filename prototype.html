Application Prototype Information

What is BonusBud? A web and mobile concept to track and complete bank signup bonuses without requiring bank logins.

Tech Used

HTML, CSS, vanilla JS. Hosted on GitHub Pages.

Core Modules

Offer catalog, requirement checklist, alerts, analytics

Data Model

User → Offers → Tasks (DD, swipes, balance) with due dates and status.

This is a non-commercial coursework prototype illustrating PO/CO outcomes: system setup, app tooling, and web deployment.
